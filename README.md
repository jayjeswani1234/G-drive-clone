# 🚀 Google Drive Clone

A modern **Google Drive Clone** built using React, designed to replicate the core functionality and UI of Google Drive. This project demonstrates file management features with a clean and responsive interface.

---

## 📌 Features

* 📁 Create and manage folders
* 📤 Upload files
* 📄 View file structure
* 🎨 Clean and responsive UI
* ⚡ Fast performance with React

## ⚙️ Installation & Setup

Follow these steps to run the project locally:

```bash
# Clone the repository
git clone https://github.com/jayjeswani1234/G-drive-clone.git

# Navigate to project folder
cd G-drive-clone

# Install dependencies
npm install

# Start the development server
npm start
```
